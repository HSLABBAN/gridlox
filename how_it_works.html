How does this work?

GRIDLOCK is solved when all 4 rows and all 4 columns contain a valid 4-letter word.

  1. You start with a 4x4 grid filled with 16 letters.
  2. The letters are already provided, but they are scrambled.
  3. Rearrange the letters so that all rows and all columns form valid 4-letter words.
  4. Each row must form a valid 4-letter word, from left-to-right OR right-to-left (frontwards or backwards).
  5. Each column must form a valid 4-letter word, from top-to-bottom OR bottom-to-top (frontwards or backwards).
